# MinecraftSkinViewer

### Install
Add it in your root build.gradle at the end of repositories:

```
allprojects {
  repositories {
    ...
    maven { url 'https://jitpack.io' }
  }
}
```

Add the dependency

```
dependencies {
  def minecraftSkinViewer = '1.0'
  implementation "com.github.luthfipun:MinecraftSkinViewer:$minecraftSkinViewer"
}
```
